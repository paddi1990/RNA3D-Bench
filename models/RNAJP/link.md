# RNAJP Resources and Downloads

## Standalone RNAJP Package (Linux)

- **RNAJP Linux Package**  
  Download the standalone RNAJP package for Linux.  
  [RNAJP_Linux.tar.bz2](http://rna.physics.missouri.edu/vfold_software_download/RNAJP_download.html) (35 MB)

- **User Manual**  
  Download the user manual for the standalone RNAJP Linux package.  
  [RNAJP_User_Manual.pdf](https://rna.physics.missouri.edu/RNAJP/RNAJP_User_Manual.pdf)

---

## RNAJP Decoy Structure Datasets

- **Decoy Structures for Junction RNAs**  
  Download the decoy structure datasets used in the RNAJP paper.  
  This archive contains **88,000 decoy structures** generated for **27 junction RNAs**.

  - Compressed size: [RNAJP_decoy_datasets.tar.bz2](https://rna.physics.missouri.edu/RNAJP/RNAJP_decoy_datasets.tar.bz2) (1.7 GB)  
  - Disk space required after extraction: ~15 GB
